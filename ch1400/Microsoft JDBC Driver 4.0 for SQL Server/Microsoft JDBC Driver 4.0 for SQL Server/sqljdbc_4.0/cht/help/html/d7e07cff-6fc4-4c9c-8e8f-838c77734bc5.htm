isNullable 方法 (SQLServerParameterMetaData)

擷取值，此值指出指定的參數中是否允許使用 Null 值。

語法

public int isNullable(int param)

參數

param

int ，指出參數索引。

傳回值

int ，指出指定之參數的 Null 屬性，可能是下列其中一個值：

ParameterMetaData.parameterNoNulls

ParameterMetaData.parameterNullable

ParameterMetaData.parameterNullabilityUnknown

例外狀況

SQLServerException

備註

這個 isNullable 方法是由 java.sql.ParameterMetaData 介面中的 isNullable 方法所指定。
