SQLServerStatement 成員

下表列出由 SQLServerStatement 類別公開的成員。

建構函式

無。

欄位

無。

繼承的欄位

名稱

描述

java.sql.Statement

CLOSE_ALL_RESULTS, CLOSE_CURRENT_RESULT, EXECUTE_FAILED, KEEP_CURRENT_RESULT, NO_GENERATED_KEYS, RETURN_GENERATED_KEYS, SUCCESS_NO_INFO

方法

名稱

描述

addBatch

加入給定的 SQL 命令至這個 SQLServerStatement 物件的目前命令清單中。

cancel

取消目前正由這個 SQLServerStatement 物件執行的 SQL 陳述式。

clearBatch

清空目前這個 SQLServerStatement 物件的 SQL 命令清單。

clearWarnings

清除這個 SQLServerStatement 物件所報告出的所有警告。

close

立刻釋放出這個 SQLServerStatement 物件的資料庫和 JDBC 資源，而非等待它們由系統自動釋放。

execute

執行給定的 SQL 陳述式，此陳述式可傳回多個結果。

executeBatch

提交批次命令到要執行的資料庫。如果所有命令都成功執行，則傳回更新計數陣列。

executeQuery

執行給定的陳述式，並傳回單一個 SQLServerResultSet 物件。

executeUpdate

執行可能是 INSERT、UPDATE、MERGE 或 DELETE 陳述式的給定 SQL 陳述式，否則會是不傳回任何項目的 SQL 陳述式，例如 SQL DDL 陳述式。

getConnection

擷取產生這個 SQLServerStatement 物件的 SQLServerConnection 物件。

getFetchDirection

擷取從資料庫資料表中提取資料列的方向，此方向是從這個 SQLServerStatement 物件產生之結果集的預設值。

getFetchSize

擷取結果集資料列數目，這個數目是從這個 SQLServerStatement 物件產生之結果集物件的預設提取大小。

getGeneratedKeys

擷取執行這個 SQLServerStatement 物件最後所建立的任何自動產生金鑰。

getMaxFieldSize

擷取最大位元組數目，此位元組可傳回做為這個 SQLServerStatement 物件所產生 SQLServerResultSet 物件內字元和二進位資料行的值。

getMaxRows

擷取最大資料列數目，即這個 SQLServerStatement 物件所產生 SQLServerResultSet 物件可以包含的資料列。

getMoreResults

移至這個 SQLServerStatement 物件的下一個結果。

getQueryTimeout

擷取 Microsoft JDBC Driver for SQL Server 將等候這個 SQLServerStatement 物件執行的秒數。

getResponseBuffering

擷取這個 SQLServerStatement 物件的回應緩衝模式。

getResultSet

擷取目前結果當做 SQLServerResultSet 物件。

getResultSetConcurrency

擷取由這個 SQLServerStatement 物件產生之 SQLServerResultSet 物件的結果集並行。

getResultSetHoldability

擷取由這個 SQLServerStatement 物件產生之 SQLServerResultSet 物件的結果集保留性。

getResultSetType

擷取由這個 SQLServerStatement 物件產生之 SQLServerResultSet 物件的結果集類型。

getUpdateCount

透過更新計數，擷取目前結果。

getWarnings

擷取由呼叫這個 SQLServerStatement 物件所報告的第一個警告。

isClosed

指出這個 SQLServerStatement 物件是否已遭關閉。

isPoolable

傳回值，這個值指出陳述式是否可以加入至使用者提供的陳述式集區。

isWrapperFor

指出這個陳述式物件是否為指定之介面的包裝函式。

setCursorName

設定 SQL 資料指標名稱為給定的 String，此字串將由後續 execute 方法使用。

setEscapeProcessing

設定逸出處理模式。

setFetchDirection

提供 JDBC Driver 提示，當做處理結果集資料列時應遵循的方向。

setFetchSize

提供 JDBC Driver 提示，當做在需要更多資料列時應該從資料庫中提取的資料列數目。

setMaxFieldSize

設定 SQLServerResultSet 資料行儲存字元或二進位值的最大位元組數目限制為給定的位元組數目。

setMaxRows

將任何 SQLServerResultSet 物件可以包含的最大資料列數目限制設定為給定的數目。

setPoolable

要求共用或不共用陳述式。

setQueryTimeout

設定驅動程式將等候這個 SQLServerStatement 物件執行指定秒數的秒數。

setResponseBuffering

設定這個 SQLServerStatement 物件的回應緩衝模式為不區分大小寫 String full 或 adaptive。

unwrap

傳回可實作指定之介面的物件，此介面可允許存取 Microsoft JDBC Driver for SQL Server 特有的方法。
