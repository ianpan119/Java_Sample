SSTRANSTIGHTLYCPLD 欄位 (SQLServerXAResource)

用來允許緊密結合的 XA 交易，這些交易擁有不同的 XA 分支交易識別碼 (XID)，但是擁有相同的全域交易識別碼 (GTRID)。

語法

public static final int SSTRANSTIGHTLYCPLD

欄位值

int 值 32768。

備註

每一筆交易都是由 XA 分支交易識別碼 (XID) 和全域交易識別碼 (GTRID) 所識別。為了讓應用程式使用緊密結合的 XA 交易 (這些交易擁有不同的 XID，但是有相同的 GTRID)，您必須在 XAResource.start 方法的 flags 參數上設定 SSTRANSTIGHTLYCPLD。如需如何使用此旗標的詳細資訊，請參閱了解 XA 交易。
