SQLServerConnection 類別

代表與 MicrosoftSQL Server 資料庫之間的 JDBC 連接。

封裝：com.microsoft.sqlserver.jdbc

實作：ISQLServerConnection、java.io.Serializable

語法

public class SQLServerConnection

備註

SQLServerConnection 支援 JDBC 連接共用，而且可以是實體 JDBC 連接或邏輯 JDBC 連接。SQLServerConnection 會管理從它建立之所有陳述式的交易控制，而且可以參與透過 XAResource 配接器所管理的 XA 分散式交易。

SQLServerConnection 會管理已備妥之陳述式控制代碼的集區。備妥的陳述式只需準備一次，通常就可以針對其參數使用不同的資料值來執行多次。備妥的陳述式也會在不同的邏輯 (共用) 連接關閉之間維護。

SQLServerConnection 不具備執行緒安全性。但是，從單一連接建立的多個陳述式可以同時在並行執行緒內處理。

此類別支援解除包裝成為 SQLServerConnection 類別、 java.sql.connection 介面與 ISQLServerConnection 介面。如需詳細資訊，請參閱＜包裝函式與介面＞。
