getMinorVersion 方法 (SQLServerDriver)

傳回 Microsoft JDBC Driver for SQL Server 的次要版本號碼。

語法

public int getMinorVersion()

傳回值

int，包含 JDBC 驅動程式的次要版本號碼。

備註

這個 getMinorVersion 方法是由 java.sql.Driver 介面中的 getMinorVersion 方法所指定。
