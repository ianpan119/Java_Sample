getMoreResults 方法 (int)

移至這個 SQLServerStatement 物件的下一個結果，並根據給定模式所指定的指示來處理目前開啟的任何結果集物件。

語法

public final boolean getMoreResults(int mode)

參數

mode

int ，指出如何處理目前開啟的結果集物件。必須是下面其中一個常數：

CLOSE_CURRENT_RESULT

KEEP_CURRENT_RESULT (JDBC 驅動程式不支援)

CLOSE_ALL_RESULTS

傳回值

如果傳回的結果是結果集則為 true，否則為 false。

例外狀況

SQLServerException

備註

這個 getMoreResults 方法是由 java.sql.Statement 介面中的 getMoreResults 方法所指定。

如果在擷取結果之前呼叫 getMoreResults 方法，它的行為會如同 mode 引數所指定，並移到下一個結果。

JDBC 驅動程式不支援使用 KEEP_CURRENT_RESULT 常數。如果使用此常數的話，將擲回例外狀況。
