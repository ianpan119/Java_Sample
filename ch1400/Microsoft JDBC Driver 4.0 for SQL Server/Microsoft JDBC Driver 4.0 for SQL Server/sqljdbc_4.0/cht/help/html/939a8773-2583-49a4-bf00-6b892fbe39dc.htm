JDBC Driver 概觀

Microsoft JDBC Driver for SQL Server 是與 Java Database Connectivity (JDBC) 4.0 相容的驅動程式，它可讓您透過強固的方式存取 Microsoft SQL Server 資料庫的資料。此 JDBC 驅動程式可以存取許多 SQL Server 2005 所引進的新功能，包括資料庫鏡像、XML、使用者定義和大數值資料類型，並且也支援新的「快照集」交易隔離。此外，此 JDBC 驅動程式也支援使用整合式驗證。

本節中的主題會提供此 JDBC 驅動程式的一般概觀，包括使用它的系統需求、使用的方式以及可取得詳細資訊的位置。
