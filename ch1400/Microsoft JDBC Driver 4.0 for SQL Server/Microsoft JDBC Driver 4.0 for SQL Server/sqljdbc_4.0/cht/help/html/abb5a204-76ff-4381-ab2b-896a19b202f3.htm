getMaxRowSize 方法 (SQLServerDatabaseMetaData)

擷取這個資料庫允許在單一資料列中使用的最大位元組數目。

語法

public int getMaxRowSize()

傳回值

int ，指出允許的最大位元組數目。

例外狀況

SQLServerException

備註

這個 getMaxRowSize 方法是由 java.sql.DatabaseMetaData 介面中的 getMaxRowSize 方法所指定。
