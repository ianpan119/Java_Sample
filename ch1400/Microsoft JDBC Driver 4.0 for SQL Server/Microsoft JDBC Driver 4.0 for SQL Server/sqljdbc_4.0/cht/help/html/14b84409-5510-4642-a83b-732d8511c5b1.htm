setObject 方法 (java.lang.String, java.lang.Object)

使用給定物件，設定指定之參數的值。

語法

public void setObject(java.lang.String sCol,
                      java.lang.Object o)

參數

sCol

String ，包含參數名稱。

o

Object 值。

例外狀況

SQLServerException

備註

這個 setObject 方法是由 java.sql.CallableStatement 介面中的 setObject 方法所指定。

如果給定 NULL，這個方法會將指定的參數轉換成 CHAR，然後再將它傳送給資料庫。如果此參數宣告為 binary、varbinary 或 image SQL 類型，則當執行陳述式時將會擲回例外狀況。

從 SQL Server JDBC 驅動程式 3.0 開始，這個方法的行為是由 sendTimeAsDatetime 連接屬性 (設定連接屬性) 和 SQLServerDataSource.setSendTimeAsDatetime 所修改。

如需詳細資訊，請參閱＜設定 java.sql.Time 值如何傳送給伺服器＞。
