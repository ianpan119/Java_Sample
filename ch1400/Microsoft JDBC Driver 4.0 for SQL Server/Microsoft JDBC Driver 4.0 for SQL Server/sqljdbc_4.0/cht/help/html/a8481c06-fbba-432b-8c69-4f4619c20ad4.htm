SQLServerPreparedStatement 類別

代表 JDBC 備妥之陳述式功能的基本實作。

封裝：com.microsoft.sqlserver.jdbc

擴充：SQLServerStatement

實作：ISQLServerPreparedStatement

語法

public class SQLServerPreparedStatement

備註

SQLServerPreparedStatement 提供的方法可讓您提供參數，例如任何原生 Java 類型及許多 Java 物件類型。SQLServerPreparedStatement 會使用 SQL Serversp_prepare 預存程序來準備陳述式，然後在後續每一次執行該陳述式時重複使用傳回的陳述式控制代碼，通常會使用使用者提供的不同參數。

SQLServerPreparedStatement 支援批次處理，也就是一組備妥的陳述式會在單一資料庫往返中執行，以改善執行階段效能。

此類別支援解除包裝成為 SQLServerPreparedStatement 類別、ISQLServerPreparedStatement 介面、 java.sql.PreparedStatement 介面與 SQLServerStatement 用來解除包裝所支援的類別和介面。如需詳細資訊，請參閱＜包裝函式與介面＞。
