updateBytes 方法 (java.lang.String, byte[])

透過給定的資料行名稱，使用 byte 值的陣列來更新指定的資料行。

語法

public void updateBytes(java.lang.String columnName,
                        byte[] x)

參數

columnName

String，包含資料行名稱。

x

byte 值的陣列。

例外狀況

SQLServerException

備註

這個 updateBytes 方法是由 java.sql.ResultSet 介面中的 updateBytes 方法所指定。

在舊版的 Microsoft JDBC Driver for SQL Server 中，您可以使用 SQLServerResultSet.updateBytes 來轉換位元組陣列與 SQL Server 資料類型 date、time、datetime2 或 datetimeoffset 之間的值。現在，搭配這些資料類型使用這個方法將會引發例外狀況，指出不支援轉換。
