使用 JDBC Driver 處理中繼資料

Microsoft JDBC Driver for SQL Server 可用許多不同的方式來處理 SQL Server 資料庫的中繼資料。JDBC 驅動程式可用來取得有關資料庫的中繼資料、結果集或參數。

JDBC 驅動程式提供三種類別來擷取 SQL Server 資料庫的中繼資料：

  • SQLServerDatabaseMetaData，用來傳回目前連接之資料庫的相關資訊。

  • SQLServerResultSetMetaData，用來傳回結果集的相關資訊。

  • SQLServerParameterMetaData，用來傳回已準備和可呼叫陳述式的參數相關資訊。

這一節的主題說明如何使用這三個中繼資料類型來使用 SQL Server 資料庫的中繼資料。

從應用程式效能方面來看，本節所討論的中繼資料方法一般而言成本較高，因此使用時要小心。
