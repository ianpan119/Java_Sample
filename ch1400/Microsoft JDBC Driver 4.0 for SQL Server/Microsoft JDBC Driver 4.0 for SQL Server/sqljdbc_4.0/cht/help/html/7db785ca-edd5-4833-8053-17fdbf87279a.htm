方法

名稱

描述

free

這個方法會釋放 CLOB 物件並且釋出它所保留的資源。

getAsciiStream

具體化此 Clob 當做 ASCII 資料流。

getCharacterStream

傳回 Clob 資料當做 java.io.Reader 物件或字元資料流。

getSubString

依據指定開始的位置和要複製的字元數目，傳回 Clob 中所指定子字串的副本。

length

傳回 Clob 中的字元數。

position

依據指定的開始位置，傳回所指定 Clob 物件或 Clob 中之子字串的字元位置。

setAsciiStream

傳回資料流，此資料流將用於從指定位置開始將 ASCII 字元寫入到這個 Clob。

setCharacterStream

傳回資料流，此資料流將用於從指定位置開始將 Unicode 字元資料流寫入到這個 Clob。

setString

從指定位置開始，寫入給定的字串到這個 Clob。

truncate

截斷 Clob 成為指定的長度。
