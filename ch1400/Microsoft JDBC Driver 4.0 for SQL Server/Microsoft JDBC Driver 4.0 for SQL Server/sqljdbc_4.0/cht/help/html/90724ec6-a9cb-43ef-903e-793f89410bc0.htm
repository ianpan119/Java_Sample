保護 JDBC Driver 應用程式

加強 Microsoft JDBC Driver for SQL Server 應用程式的安全性不僅是避免常犯的編碼錯誤而已。存取資料的應用程式有許多攻擊者可以用於擷取、操作或破壞敏感性資料的潛在錯誤點。了解安全性的各個層面相當重要，包括在應用程式的設計階段處理威脅模型到其最終的部署，以及繼續持續的維護。

本節中的主題描述一些常見的安全性考量，包括連接字串、驗證使用者輸入，以及一般應用程式安全性。
