getSendTimeAsDatetime 方法 (SQLServerDataSource)

這個方法是在 SQL Server JDBC 驅動程式 3.0 中所加入。

傳回 sendTimeAsDatetime 連接屬性的設定。

語法

public boolean getSendTimeAsDatetime();

傳回值

如果 java.sql.Time 值將會當做 SQL Serverdatetime 類型傳送給伺服器則為 true。如果 java.sql.Time 值將會當做 SQL Servertime 類型傳送給伺服器則為 false。

備註

如需 sendTimeAsDatetime 連接屬性的詳細資訊，請參閱設定連接屬性。

SQLServerDataSource.setSendTimeAsDatetime 可讓您以程式設計方式設定 sendTimeAsDatetime 連接屬性。

如需詳細資訊，請參閱＜設定 java.sql.Time 值如何傳送給伺服器＞。
