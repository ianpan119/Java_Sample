SQLServerDataSource 成員

下表列出由 SQLServerDataSource 類別公開的成員。

建構函式

名稱

描述

SQLServerDataSource ()

初始化 SQLServerDataSource 類別的新執行個體。

欄位

無。

繼承的欄位

無。

方法

名稱

說明

getApplicationIntent

傳回 applicationIntent 連接屬性的值。

getApplicationName

傳回應用程式名稱。

getConnection

嘗試與這個 SQLServerDataSource 物件所代表的資料來源建立連接。

getDatabaseName

傳回資料庫名稱。

getEncrypt

傳回 Boolean 值，這個值會指出是否啟用 encrypt 屬性。

getDescription

傳回資料來源的描述。

getFailoverPartner

傳回用於資料庫鏡像組態的容錯移轉伺服器名稱。

getHostNameInCertificate

傳回用於驗證 SQL Server 安全通訊端層 (SSL) 憑證的主機名稱。

getInstanceName

傳回 SQL Server 執行個體名稱。

getLastUpdateCount

傳回 boolean 值，這個值會指出是否啟用 lastUpdateCount 屬性。

getLockTimeout

傳回 int 值，這個值指出資料庫在報告鎖定逾時之前所要等候的毫秒數。

getLoginTimeout

傳回這個 SQLServerDataSource 物件在嘗試建立連接時的等候秒數。

getLogWriter

傳回要用於所有記錄和追蹤訊息的字元輸出資料流。

getMultiSubnetFailover

傳回 multiSubnetFailover 連接屬性的值。

getPacketSize

傳回用來與 SQL Server 通訊的目前網路封包大小 (以位元組為單位)。

getPortNumber

傳回目前用來與 SQL Server 通訊的通訊埠編號。

getReference

傳回這個 SQLServerDataSource 物件的參考。

getResponseBuffering

傳回這個 SQLServerDataSource 物件的回應緩衝模式。

getSelectMethod

傳回使用這個 SQLServerDataSource 物件所建立之所有結果集所使用的預設資料指標類型。

getSendStringParametersAsUnicode

傳回 Boolean 值，這個值指出是否啟用以 UNICODE 格式傳送字串參數到伺服器。

getSendTimeAsDatetime

傳回 SendTimeAsDatetime 連接屬性的設定。

getServerName

傳回正在執行 SQL Server 的電腦名稱。

getTrustServerCertificate

傳回 Boolean 值，這個值會指出是否啟用 trustServerCertificate 屬性。

getTrustStore

傳回憑證 trustStore 檔案的路徑 (包括檔案名稱)。

getURL

傳回用來連接到資料來源的 URL。

getUser

傳回用來連接到資料來源的使用者名稱。

getUseSQLServerBaseDate

傳回 useSQLServerBaseDate 連接屬性的設定。

getWorkstationID

傳回用來連接到資料來源之用戶端電腦的名稱。

getXopenStates

傳回 Boolean 值，這個值指出是否啟用將 SQL 狀態轉換成 XOPEN 標準狀態。

isWrapperFor

指出這個資料來源物件是否為指定之介面的包裝函式。

setApplicationIntent

設定 applicationIntent 連接屬性的值。

setApplicationName

設定應用程式名稱。

setAuthenticationScheme

表示您希望應用程式使用的整合式安全性種類。

setDatabaseName

設定要連接的資料庫名稱。

setDescription

設定資料來源的描述。

setEncrypt

設定 Boolean 值，這個值會指出是否啟用 encrypt 屬性。

setFailoverPartner

設定用於資料庫鏡像組態的容錯移轉伺服器名稱。

setHostNameInCertificate

設定用於驗證 SQL Server 安全通訊端層 (SSL) 憑證的主機名稱。

setInstanceName

設定 SQL Server 執行個體名稱。

setIntegratedSecurity

設定 Boolean 值，這個值會指出是否啟用 integratedSecurity 屬性。

setLastUpdateCount

設定 Boolean 值，這個值會指出是否啟用 lastUpdateCount 屬性。

setLockTimeout

設定 int 值，這個值指出資料庫在報告鎖定逾時之前要等候的毫秒數。

setLoginTimeout

設定這個 SQLServerDataSource 物件在嘗試建立連接時的等候秒數。

setLogWriter

設定要用於所有記錄和追蹤訊息的字元輸出資料流。

setMultiSubnetFailover

設定 multiSubnetFailover 連接屬性的值。

setPacketSize

設定用來與 SQL Server 通訊的網路封包大小 (以位元組為單位)。

setPassword

設定用來連接到 SQL Server 的密碼。

setPortNumber

設定目前用來與 SQL Server 通訊的連接埠編號。

setResponseBuffering

設定連接的回應緩衝模式，此連接是使用這個 SQLServerDataSource 物件所建立的。

setSelectMethod

設定使用這個 SQLServerDataSource 物件所建立之所有結果集所使用的預設資料指標類型。

setSendStringParametersAsUnicode

設定 Boolean 值，這個值指出是否啟用以 UNICODE 格式傳送字串參數到伺服器。

setSendTimeAsDatetime

指定如何將 java.sql.Time 值傳送給伺服器。

setServerName

設定正在執行 SQL Server 的電腦名稱。

setTrustServerCertificate

設定 Boolean 值，這個值會指出是否啟用 trustServerCertificate 屬性。

setTrustStore

設定憑證 trustStore 檔案的路徑 (包括檔案名稱)。

setTrustStorePassword

設定用來檢查 trustStore 資料完整性的密碼。

setURL

設定用來連接到資料來源的 URL。

setUser

設定用來連接到資料來源的使用者名稱。

setWorkstationID

設定用來連接到資料來源之用戶端電腦的名稱。

setXopenStates

設定 Boolean 值，這個值指出是否啟用將 SQL 狀態轉換成 XOPEN 標準狀態。

unwrap

傳回可實作指定之介面的物件，此介面可允許存取 Microsoft JDBC Driver for SQL Server 特有的方法。
