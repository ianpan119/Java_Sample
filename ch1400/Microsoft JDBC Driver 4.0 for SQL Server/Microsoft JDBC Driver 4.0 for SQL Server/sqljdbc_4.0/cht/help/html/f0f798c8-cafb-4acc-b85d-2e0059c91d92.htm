setPoolable 方法 (ISQLServerStatement)

要求共用或不共用陳述式。

語法

public void setPoolable(boolean poolable) throws SQLException

參數

poolable

如果為 true，則表示應該共用陳述式。如果為 false，則表示不應該共用陳述式。

例外狀況

SQLServerException

備註

poolable 參數中指定的值是陳述式集區實作的提示，可指出應用程式是否想要共用陳述式。陳述式集區管理員會決定它是否使用該提示。

陳述式的集區值會套用到驅動程式所實作的內部陳述式快取，以及應用程式伺服器和其他應用程式所實作的外部陳述式快取。

根據預設，建立 SQLServerStatement 物件時無法共用，而建立 SQLServerPreparedStatement 和 SQLServerCallableStatement 物件時則可以共用。

如果這個方法在關閉的陳述式上呼叫，則會擲回 SQLServerException。

isPoolable 傳回值，指出是否可共用該物件。
