方法

名稱

描述

free

這個方法會釋放 BLOB 物件並且釋出它所保留的資源。

getBinaryStream

傳回輸入資料流，以便從 BLOB 讀取資料。

getBytes

取得 BLOB 資料來當做位元組陣列。

length

傳回 BLOB 物件中的位元組數目。

position

依據給定模式和開始索引，傳回指定模式在 BLOB 中的位置。

setBinaryStream

擷取可用來寫入至 BLOB 值的資料流。

setBytes

寫入給定位元組陣列至從給定位置開始的 BLOB，然後傳回寫入的位元組數目。

truncate

依給定長度截斷 BLOB。
