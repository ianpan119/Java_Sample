getColumnLabel 方法 (SQLServerResultSetMetaData)

取得指定之資料行之列印輸出和顯示中所使用的建議標題。

語法

public java.lang.String getColumnLabel(int column)

參數

column

int ，指出此資料行索引。

傳回值

String ，包含資料行的標題。

例外狀況

SQLServerException

備註

這個 getColumnLabel 方法是由 java.sql.ResultSetMetaData 介面中的 getColumnLabel 方法所指定。

這個方法會傳回資料行的別名名稱。如果沒有別名名稱，這個方法會傳回資料行名稱。
