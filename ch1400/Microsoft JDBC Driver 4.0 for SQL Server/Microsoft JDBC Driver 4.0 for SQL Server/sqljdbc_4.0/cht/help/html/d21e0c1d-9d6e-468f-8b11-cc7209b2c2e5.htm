getTime 方法 (int, java.util.Calendar) (ISQLServerResultSet)

使用 Java 程式語言，並透過給定的 Calendar 物件，擷取這個 SQLServerResultSet 物件中目前資料列內所指定之資料行索引的值來當做 java.sql.Time 物件。

語法

public java.sql.Time getTime(int columnIndex,
                             java.util.Calendar cal)

參數

columnIndex

int ，指出此資料行索引。

cal

Calendar 物件。

傳回值

Time 物件。

例外狀況

SQLServerException

備註

這個 getTime 方法是由 java.sql.ResultSet 介面中的 getTime 方法所指定。

這個方法會傳回 SQL Server datetime 或 smalldatetime 資料類型中的有效時間部分，而日期部分是設定成 Java 基準日期 1970/01/01 (根據提供的月曆時區)。
