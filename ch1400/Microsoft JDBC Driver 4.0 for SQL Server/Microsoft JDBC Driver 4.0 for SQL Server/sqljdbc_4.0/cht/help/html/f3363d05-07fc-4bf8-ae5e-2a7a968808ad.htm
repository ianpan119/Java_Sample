設定資料來源屬性

資料來源為在 Java Platform Enterprise Edition (Java EE) 環境中建立 JDBC 連接所慣用的機制。資料來源提供連接、共用連接及分散式連接，而未在 Java 程式碼內將連接屬性寫入程式碼。所有 Microsoft JDBC Driver for SQL Server 資料來源可分別使用適當的 setter 與 getter 方法，以設定或取得任何屬性的值。

Java EE 產品 (例如應用程式伺服器及 servlet/JSP 引擎) 通常會讓您設定資料庫存取的資料來源。只要組態能讓您將屬性輸入成 property=value 組，即可指定在＜設定連接屬性＞主題中所列的任一屬性。

如需有關 SQL Server 資料來源的詳細資訊，請參閱 SQLServerDataSource 類別。如需如何使用 SQLServerDataSource 類別連接到 SQL Server 資料庫的範例，請參閱＜資料來源範例＞。
