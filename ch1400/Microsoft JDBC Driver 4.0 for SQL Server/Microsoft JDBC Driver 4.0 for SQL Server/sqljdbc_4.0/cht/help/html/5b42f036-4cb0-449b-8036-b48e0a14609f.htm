position 方法 (SQLServerNClob)

依據指定的開始位置，擷取所指定 NClob 物件或 NClob 中之子字串的字元位置。

多載清單

名稱

描述

position 方法 (java.sql.NClob, long)

擷取字元位置，也就是指定之 NClob 物件 searchstr 在這個 NClob 物件中的出現位置。

position 方法 (java.lang.String, long) (SQLServerNClob)

擷取字元位置，也就是指定之子字串 searchstr 在 NCLOB 值中的出現位置，而此值會以這個 NClob 物件表示。
