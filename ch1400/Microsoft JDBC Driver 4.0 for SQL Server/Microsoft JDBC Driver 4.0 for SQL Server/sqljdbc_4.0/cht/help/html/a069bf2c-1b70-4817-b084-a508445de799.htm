欄位

名稱

說明

SSTRANSTIGHTLYCPLD

用來允許緊密結合的 XA 交易，這些交易擁有不同的 XA 分支交易識別碼 (XID)，但是擁有相同的全域交易識別碼 (GTRID)。

方法

名稱

說明

commit

認可給定 Xid 物件所指定的全域交易。

end

結束代表交易分支執行的工作。

forget

告知資源管理員要忘記以啟發方式完成的交易分支。

getTransactionTimeout

取得為這個 SQLServerXAResource 物件設定的目前交易逾時值。

isSameRM

判斷目標物件所表示的資源管理員執行個體是否與給定 XAResource 物件所表示的資源管理員執行個體相同。

prepare

要求資源管理員準備給定 Xid 物件所指定之交易的交易認可。

recover

從資源管理員取得備妥的交易分支清單。

rollback

要求資源管理員回復之前代表交易分支所完成的工作。

setTransactionTimeout

設定這個 SQLServerXAResource 物件的目前交易逾時值。

start

代表 Xid 物件中指定的交易分支開始工作。
