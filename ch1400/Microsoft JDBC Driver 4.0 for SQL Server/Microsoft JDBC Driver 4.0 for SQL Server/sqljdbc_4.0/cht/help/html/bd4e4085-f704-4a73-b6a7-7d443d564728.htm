prepareStatement 方法 (ISQLServerConnection)

建立 SQLServerPreparedStatement 物件，以便將參數化 SQL 陳述式傳送到資料庫。

多載清單

名稱

說明

prepareStatement (java.lang.String)

建立 SQLServerPreparedStatement 物件，以便將參數化 SQL 陳述式傳送到資料庫。

prepareStatement (java.lang.String, int)

建立 SQLServerPreparedStatement 物件，以便將參數化 SQL 陳述式傳送到資料庫，並且能夠擷取自動產生的索引鍵。

prepareStatement (java.lang.String, int[])

建立 SQLServerPreparedStatement 物件，以便將參數化 SQL 陳述式傳送到資料庫，並且能夠傳回給定陣列所指定之自動產生的索引鍵。

prepareStatement (java.lang.String, int, int)

建立 SQLServerPreparedStatement 物件，它會產生具有給定類型和並行的 SQLServerResultSet 物件。

prepareStatement (java.lang.String, int, int, int)

建立 SQLServerPreparedStatement 物件，它會產生具有給定類型、並行和保留性的 SQLServerResultSet 物件。

prepareStatement (java.lang.String, java.lang.String[])

建立 SQLServerPreparedStatement 物件，以便將參數化 SQL 陳述式傳送到資料庫。
