setUnicodeStream 方法 (ISQLServerPreparedStatement)

將指定的參數號碼設定為給定的輸入資料流，其中將包含指定的位元組數目。

JDBC 規格中已經取代這個方法，呼叫此方法將會擲回「未實作」例外狀況。

語法

public final void setUnicodeStream(int n,
                                   java.io.InputStream x,
                                   int length)

參數

n

int ，指出參數編號。

x

InputStream 物件。

length

位元組數目。

例外狀況

SQLServerException

備註

這個 setUnicodeStream 方法是由 java.sql.PreparedStatement 介面中的 setUnicodeStream 方法所指定。
