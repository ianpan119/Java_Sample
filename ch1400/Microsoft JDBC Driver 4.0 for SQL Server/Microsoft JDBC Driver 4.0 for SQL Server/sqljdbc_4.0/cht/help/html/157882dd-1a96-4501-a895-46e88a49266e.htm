setClob 方法 (int, java.io.Reader, long)

設定指定的參數為指定的 Reader 物件，該物件長度為給定的字元數。

語法

public final void setClob(int parameterIndex,
                          java.io.Reader reader,
                          long length)

參數

parameterIndex

int，指出參數索引。

reader

Reader 物件。

length

long，指出參數值中的字元數目。

備註

這個 setClob 方法是由 java.sql.PreparedStatement 介面中的 setClob 方法所指定。

例外狀況

SQLServerException
