getUnicodeStream 方法 (ISQLServerResultSet)

擷取這個 SQLServerResultSet 物件中目前資料列內指定之資料行的值來當做 Unicode 字元資料流。

JDBC 規格中已經取代這個方法，呼叫此方法將會擲回「未實作」例外狀況。您應該改用 getCharacterStream 方法。
