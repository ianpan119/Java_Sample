registerOutParameter 方法 (java.lang.String, int, java.lang.String)

向給定 JDBC 類型和類別名稱的指定名稱註冊 OUT 參數。

語法

public void registerOutParameter(java.lang.String s,
                                 int n,
                                 java.lang.String s1)

參數

s

String ，包含參數名稱。

n

JDBC 類型程式碼，如 java.sql.Types 中所定義。

s1

String ，包含完整的 SQL 類型名稱。

例外狀況

SQLServerException

備註

這個 registerOutParameter 方法是由 java.sql.CallableStatement 介面中的 registerOutParameter 方法所指定。
