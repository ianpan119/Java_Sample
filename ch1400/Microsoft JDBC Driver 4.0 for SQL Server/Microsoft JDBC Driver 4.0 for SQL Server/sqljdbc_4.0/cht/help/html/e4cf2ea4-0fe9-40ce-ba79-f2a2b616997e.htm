compareTo 方法 (DateTimeOffset)

根據 GMT 的時間，比較此 DateTimeOffset 物件與其他 DateTimeOffset 物件。

語法

public int compareTo(DateTimeOffset other)

參數

您想要與目前執行個體比較的 DateTimeOffset 值。

傳回值

下表描述此方法的傳回值。

傳回值

描述

0

兩個 DateTimeOffset 物件都表示相同的時間點。

負數

此 DateTimeOffset 物件表示 other 前的時間點。

正數

此 DateTimeOffset 物件表示 other 後的時間點。

備註

當兩個 DateTimeOffset 物件有相同的 GMT 時間時，根據位移，物件就沒有其他順序。
