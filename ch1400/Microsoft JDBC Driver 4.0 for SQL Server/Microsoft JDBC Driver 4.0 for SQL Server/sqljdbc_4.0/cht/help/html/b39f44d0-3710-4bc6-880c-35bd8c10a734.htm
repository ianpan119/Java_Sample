使用資料類型 (JDBC)

Microsoft JDBC Driver for SQL Server 的主要功能就是可讓 Java 開發人員存取 SQL Server 資料庫中包含的資料。為了完成此項工作，JDBC 驅動程式會調解 SQL Server 資料類型和 Java 語言類型與物件之間的轉換。

如需 SQL Server 及 JDBC 驅動程式資料類型的詳細討論，包括其差異和它們如何轉換為 Java 語言資料類型，請參閱＜了解 JDBC Driver 資料類型＞。

為了能夠使用 SQL Server 資料類型，JDBC 驅動程式提供了 get<Type> 和 set<Type> 方法給 SQLServerPreparedStatement 和 SQLServerCallableStatement 類別，它也提供 get<Type> 和 update<Type> 方法給 SQLServerResultSet 類別。使用哪一種方法視您正要使用的資料類型而定，以及是否會使用結果集或查詢。

本節的主題說明如何使用 JDBC 驅動程式資料類型，在 Java 應用程式中存取 SQL Server 資料。
