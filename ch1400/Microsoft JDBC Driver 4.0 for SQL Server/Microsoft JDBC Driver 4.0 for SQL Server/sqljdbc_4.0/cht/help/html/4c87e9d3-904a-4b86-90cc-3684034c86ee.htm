SQLServerPooledConnection 類別

代表連接共用中的實體資料庫連接。

封裝：com.microsoft.sqlserver.jdbc

擴充：java.lang.Object

實作：javax.sql.PooledConnection

語法

public final class SQLServerPoolingDataSource

備註

SQLServerPooledConnection 類別會提供方法給連接共用管理員來管理連接共用。應用程式通常不會直接將這些連接具現化。
