isWritable 方法 (SQLServerResultSetMetaData)

指出寫入至指定之資料行的作業是否可能成功。

語法

public boolean isWritable(int column)

參數

column

int ，指出此資料行索引。

傳回值

如果資料行寫入成功則為 true，否則為 false。

例外狀況

SQLServerException

備註

這個 isWritable 方法是由 java.sql.ResultSetMetaData 介面中的 isWritable 方法所指定。
