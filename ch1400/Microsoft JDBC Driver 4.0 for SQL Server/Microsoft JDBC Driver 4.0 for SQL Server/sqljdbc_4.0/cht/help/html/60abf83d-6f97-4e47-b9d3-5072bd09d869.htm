getDateTimeOffset(int) (SQLServerResultSet)

這個方法是在 Microsoft SQL Server JDBC Driver 3.0 中所加入。

使用 Java 程式語言，並配合給定的參數索引來擷取指定之資料行的值當做 DateTimeOffset 類別 物件。

語法

public microsoft.sql.DateTimeOffset getDateTimeOffset(int columnIndex)

參數

columnIndex

資料行序數。

傳回值

DateTimeOffset 類別 物件。

例外狀況

SQLServerException

備註

您可以使用 SQLServerResultSet.updateDateTimeOffset 來更新 DateTimeOffset 類別 值。
