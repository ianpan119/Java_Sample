使用 JDBC 驅動程式連接到 SQL Server

您將利用 Microsoft JDBC Driver for SQL Server 進行的其中一項最基本動作就是連接到 SQL Server 資料庫。所有與資料庫的互動都是經由 SQLServerConnection 物件發生，而由於 JDBC 驅動程式擁有相當平面的架構，因此幾乎所有令人關注的行為都會用到 SQLServerConnection 物件。

如果 SQL Server 只接聽 IPv6 通訊埠，請設定 java.net.preferIPv6Addresses 系統屬性，以確保使用 IPv6 而不是 IPv4 來連接 SQL Server：

System.setProperty("java.net.preferIPv6Addresses", "true");

本節的主題說明如何建立及使用 SQL Server 資料庫的連接。
