findColumn 方法 (ISQLServerResultSet)

擷取與這個 SQLServerResultSet 物件中給定資料行名稱相符合之第一個資料行的索引。

語法

public int findColumn(java.lang.String columnName)

參數

columnName

String ，包含資料行的名稱。

傳回值

int ，指出此資料行索引。

例外狀況

SQLServerException

備註

這個 findColumn 方法是由 java.sql.ResultSet 介面中的 findColumn 方法所指定。

如果有多個資料行同名，findColumn 方法會傳回第一個區分大小寫的相符項目。如果沒有任何區分大小寫的相符項目，這個方法會傳回第一個不區分大小寫的相符項目。
