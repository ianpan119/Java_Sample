方法

名稱

描述

getCatalogName

取得包含所指定資料行之資料表的目錄名稱。

getColumnClassName

傳回 Java 類別的完整名稱，此類別的執行個體會在呼叫 SQLServerResultSet 的 getObject 方法來從資料行擷取值時進行製造。

getColumnCount

傳回結果集中的資料行數目。

getColumnDisplaySize

傳回所指定資料行的正常最大寬度，以字元為單位。

getColumnLabel

取得建議在指定之資料行之列印輸出和顯示中使用的標題。

getColumnName

取得所指定資料行的名稱。

getColumnType

擷取所指定資料行的 SQL 類型。

getColumnTypeName

擷取所指定資料行的資料庫特性類型名稱。

getPrecision

取得所指定資料行的小數位數。

getScale

取得所指定資料行小數點右邊的位數。

getSchemaName

取得所指定資料行的資料表結構描述名稱。

getTableName

取得所指定資料行的資料表名稱。

isAutoIncrement

指出所指定資料行是否已自動編號而形成唯讀狀態。

isCaseSensitive

指出資料行是否區分大小寫。

isCurrency

指出所指定資料行是否為貨幣 (cash) 類型值。

isDefinitelyWritable

指出寫入至所指定資料行的作業是否絕對會成功。

isNullable

指出所指定資料行中之值的 Null 屬性。

isReadOnly

指出所指定資料行是否絕對不可寫入。

isSearchable

指出所指定資料行是否能用於 SQL WHERE 子句。

isSigned

指出指定之資料行中的值是否為帶正負號的數值。

isSparseColumnSet

指出結果集中的資料行是否為疏鬆資料行集。

isWritable

指出寫入至指定之資料行的作業是否可能成功。
