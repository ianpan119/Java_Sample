使用 SQL 陳述式

使用 Microsoft JDBC Driver for SQL Server 及內嵌 SQL 陳述式來處理 SQL Server 資料庫中的資料時，有不同的類別可供您使用。使用哪一種類別視您想要執行的 SQL 陳述式類型而定。

如果您的 SQL 陳述式不含任何 IN 參數，請使用 SQLServerStatement 類別，但如果它確實含有 IN 參數，則請使用 SQLServerPreparedStatement 類別。

如果您需要使用同時包含 IN 與 OUT 參數的 SQL 陳述式，您必須將它們實作為預存程序，並使用 SQLServerCallableStatement 類別呼叫它們。如需使用預存程序的詳細資訊，請參閱＜搭配預存程序使用陳述式＞。
