setNCharacterStream 方法 (java.lang.String, java.io.Reader)

將指定的參數設定為指定的 Reader 物件。

語法

public final void setNCharacterStream(java.lang.String parameterName,
                       java.io.Reader value)

參數

parameterName

String，指出參數名稱。

value

Reader 物件。

例外狀況

SQLServerException

備註

這個 setNCharacterStream 方法是由 java.sql.CallableStatement 介面中的 setNCharacterStream 方法所指定。

這個方法也可以用於 NCHAR、NVARCHAR、NTEXT 和 XML 等資料類型。
