JDBC Driver API 參考

Microsoft JDBC Driver for SQL Server 提供一種 API，此 API 可在 Java 程式碼中用來與 Microsoft SQL Server 資料庫進行連接和互動。

如需有關使用 JDBC 驅動程式的概念性資訊，請參閱＜JDBC Driver 概觀＞。

介面

介面名稱

描述

ISQLServerCallableStatement 介面

讓您指定要呼叫的預存程序名稱，連同輸入和輸出參數。

ISQLServerConnection 介面

代表與 SQL Server 資料庫的 JDBC 連接。

SQLServerDataSource 類別

代表屬性清單，這些是使用 ISQLServerConnection 物件來連接至 SQL Server 資料庫時的特定屬性。

ISQLServerPreparedStatement

代表 JDBC 備妥之陳述式功能的基本實作。

ISQLServerResultSet

代表 JDBC 結果集。

ISQLServerStatement

代表 JDBC 陳述式功能的基本實作。

代表屬性清單，這些是使用 SQLServerConnection 物件來連接至 SQL Server 資料庫時的特定屬性。

SQLServerXAResource
