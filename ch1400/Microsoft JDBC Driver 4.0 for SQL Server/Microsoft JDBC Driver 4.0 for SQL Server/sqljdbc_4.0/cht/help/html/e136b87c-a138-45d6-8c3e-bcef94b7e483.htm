範例 JDBC 驅動程式應用程式

Microsoft JDBC Driver for SQL Server 範例應用程式會示範 JDBC 驅動程式的各種功能。此外，它們也示範當使用 JDBC 驅動程式與 SQL Server 資料庫搭配時，您可以遵循的良好程式設計作法。

所有的範例應用程式都包含在可以在本機電腦上編譯並執行的 *.java 程式碼檔案中，而且它們位於下列位置的各個子資料夾中：

<installation directory>\sqljdbc_<version>\<language>\help\samples

本節的主題說明如何設定並執行範例應用程式，並包括範例應用程式示範內容的討論。

本章節內容

主題

描述

連接及擷取資料

這些範例應用程式示範如何連接至 SQL Server 資料庫。它們也示範從 SQL Server 資料庫擷取資料的不同方法。

使用資料類型 (JDBC)

這些範例應用程式示範如何使用 JDBC 驅動程式資料類型方法，來使用 SQL Server 資料庫中的資料。

使用結果集

這些範例應用程式示範如何使用結果集，來處理 SQL Server 資料庫中包含的資料。

使用大型資料

這些範例應用程式會示範如何使用適應性緩衝來擷取 SQL Server 資料庫中的大數值資料，而沒有伺服器資料指標的負擔。
