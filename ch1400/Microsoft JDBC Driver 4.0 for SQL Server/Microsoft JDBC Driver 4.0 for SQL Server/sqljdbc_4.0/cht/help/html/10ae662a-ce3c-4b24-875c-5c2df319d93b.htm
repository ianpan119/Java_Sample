executeUpdate 方法 (SQLServerStatement)

執行可能是 INSERT、UPDATE 或 DELETE 陳述式的給定 SQL 陳述式，否則會是不傳回任何項目的 SQL 陳述式，例如 SQL DDL 陳述式。從 Microsoft SQL Server JDBC Driver 3.0 開始，executeUpdate 將會傳回 MERGE 作業中更新的正確資料列數目。

多載清單

名稱

說明

executeUpdate (java.lang.String)

執行可能是 INSERT、UPDATE、DELETE 或 MERGE 陳述式的給定 SQL 陳述式，否則會是不傳回任何內容的 SQL 陳述式，例如 SQL DDL 陳述式。

executeUpdate (java.lang.String, int)

執行給定的 SQL 陳述式，並透過給定的旗標向 Microsoft JDBC Driver for SQL Server 發出信號，通知是否要提供由這個 SQLServerStatement 物件產生的自動產生索引鍵來進行擷取。

executeUpdate (java.lang.String, int[])

執行給定的 SQL 陳述式，並向 JDBC Driver 發出信號，通知必須提供在給定陣列中所指出的自動產生索引鍵以進行擷取。

executeUpdate (java.lang.String, java.lang.String[])

執行給定的 SQL 陳述式，並向 JDBC Driver 發出信號，通知必須提供在給定陣列中所指出的自動產生索引鍵以進行擷取。
