取得驅動程式版本

您可以用下列方式尋找已安裝的 Microsoft JDBC Driver for SQL Server 版本：

  • 呼叫 SQLServerDatabaseMetaData 方法 getDriverMajorVersion、getDriverMinorVersion 或 getDriverVersion。

  • 版本會顯示在產品散發的 readme.txt 檔中。

此外，JDBC 驅動程式名稱可以從 SQLServerDatabaseMetaData 類別上的 getDriverName 方法呼叫傳回。它會傳回 "Microsoft JDBC Driver 4.0 for SQL Server"。

下面是從 SQLServerDatabaseMetaData 類別之方法呼叫輸出的範例：

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

其中 "xxx.x" 為最終的版本號碼。
