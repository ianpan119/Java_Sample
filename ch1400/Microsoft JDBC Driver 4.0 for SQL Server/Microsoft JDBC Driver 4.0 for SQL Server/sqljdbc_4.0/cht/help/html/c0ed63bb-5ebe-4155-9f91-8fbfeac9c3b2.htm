getDouble 方法 (int)

使用 Java 程式語言，並配合給定的參數索引來擷取指定參數的值當做 double 物件。

語法

public double getDouble(int index)

參數

index

int ，指出參數索引。

傳回值

double 值。

例外狀況

SQLServerException

備註

這個 getDouble 方法是由 java.sql.CallableStatement 介面中的 getDouble 方法所指定。

這個方法會傳回所有具有 Java double 精確度的數字資料類型。
