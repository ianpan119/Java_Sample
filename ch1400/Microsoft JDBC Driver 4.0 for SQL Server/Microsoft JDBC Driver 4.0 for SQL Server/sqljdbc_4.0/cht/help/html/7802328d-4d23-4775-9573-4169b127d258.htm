了解 JDBC Driver 資料類型

Microsoft JDBC Driver for SQL Server 支援在使用 SQL Server 當做其資料庫的 Java 應用程式中，使用 JDBC 基本和進階資料類型。

JDBC 類型系統會調解 SQL Server 資料類型和 Java 語言類型與物件之間的轉換。JDBC 類型的模型是以 SQL-92 和 SQL-99 類型建立的。JDBC 驅動程式遵守 JDBC 規格，其設計在於提供可預測性與彈性之間的正確平衡。

本節中的主題描述如何使用基本和進階資料類型，以及如何將資料類型轉換為其他資料類型。
