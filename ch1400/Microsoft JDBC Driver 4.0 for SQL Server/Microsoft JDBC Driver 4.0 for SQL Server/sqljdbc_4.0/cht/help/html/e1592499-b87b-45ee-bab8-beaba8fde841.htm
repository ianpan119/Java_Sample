增進 JDBC 驅動程式的效能與可靠性

所有應用程式在應用程式開發上皆通用的一個概念即需要不斷地增進效能與可靠性。有一些技術可以用在 Microsoft JDBC Driver for SQL Server，以達成此目的。

本節的主題說明當使用 JDBC 驅動程式時，可改善應用程式效能和可靠性的各種技術。

本章節內容

主題

描述

不使用時關閉物件

說明不再需要 JDBC 驅動程式物件時，關閉它們的重要性。

管理交易大小

說明增進交易效能的技術。

使用陳述式及結果集

描述當使用 Statement 或 ResultSet 物件時改善效能的技術。

使用適應性緩衝

描述適應性緩衝功能，這項功能設計成可擷取任何種類的大數值資料，而不會造成伺服器資料指標的負擔。

疏鬆資料行

討論 JDBC 驅動程式對 SQL Server 疏鬆資料行的支援。
