executeUpdate 方法 (java.lang.String, int)

執行給定的 SQL 陳述式，並透過給定的旗標向 Microsoft JDBC Driver for SQL Server 發出信號，通知是否要提供由這個 SQLServerStatement 物件產生的自動產生索引鍵來進行擷取。

語法

public final int executeUpdate(java.lang.String sql,
                               int flag)

參數

sql

String，包含 SQL 陳述式。

flag

int 值，指出是否必須提供自動產生的索引鍵。這個值一定是下面其中一個常數：

RETURN_GENERATED_KEYS

NO_GENERATED_KEYS

傳回值

int，指出受影響的資料列數目，如果是使用 DDL 陳述式，則為 0。

例外狀況

SQLServerException

備註

這個 executeUpdate 方法是由 java.sql.Statement 介面中的 executeUpdate 方法所指定。

如果執行預存程序產生的更新計數大於一或是產生一個以上結果集，請使用 execute 方法執行預存程序。
