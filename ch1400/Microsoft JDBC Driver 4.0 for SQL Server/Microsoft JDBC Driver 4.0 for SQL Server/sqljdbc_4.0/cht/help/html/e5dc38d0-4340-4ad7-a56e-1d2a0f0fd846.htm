getPortNumber 方法 (SQLServerDataSource)

傳回目前用來與 SQL Server 通訊的通訊埠編號。

語法

public int getPortNumber()

傳回值

包含目前通訊埠編號的 int 值。

備註

通訊埠編號為 TCP/IP 通訊埠編號，當開啟與 SQL Server 之間的通訊端連接時就會使用它。如果未設定 portNumber 屬性，getPortNumber 方法會傳回預設值 1433。

setPortNumber 方法不會針對傳入的通訊埠值進行任何範圍檢查。您可以傳遞無效的通訊埠編號 (如 99999) 而不會引發錯誤。
