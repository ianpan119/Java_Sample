連接及擷取資料

當使用 Microsoft JDBC Driver for SQL Server 時，有兩種主要方法，可用於建立 SQL Server 資料庫的連接。第一種方法就是在連接 URL 中設定連接屬性，然後呼叫 DriverManager 類別的 getConnection 方法，以傳回 SQLServerConnection 物件。

如需 JDBC 驅動程式支援的連接屬性清單，請參閱＜設定連接屬性＞。

第二種方法牽涉到設定連接屬性，方法是使用 SQLServerDataSource 類別的 setter 方法，然後呼叫 getConnection 方法，以傳回 SQLServerConnection 物件。

本節的主題說明您可以連接至 SQL Server 資料庫的不同方法，而且它們也示範了擷取資料的不同技術。
