unwrap 方法 (SQLServerXADataSource)

傳回可實作指定之介面的物件，此介面可允許存取 Microsoft JDBC Driver for SQL Server 特有的方法。

語法

public <T> T unwrap(Class<T> iface)

參數

iface

T 類型的類別，可定義介面。

傳回值

物件，可實作指定的介面。

例外狀況

SQLServerException

備註

unwrap 方法是由已自 JDBC 4.0 規格中導入的 java.sql.Wrapper 介面所定義。

應用程式可能必須存取 Microsoft JDBC Driver for SQL Server 特有的 JDBC API 延伸模組。在類別公開供應商延伸模組的情況下，unwrap 方法支援解除包裝成為這個物件可延伸的公用類別。

SQLServerXADataSource 類別會擴充 SQLServerConnectionPoolDataSource 類別，而其是由 SQLServerDataSource 類別擴充所得到的類別。當呼叫這個方法時，物件會解除包裝成為下列類別：SQLServerDataSource、SQLServerConnectionPoolDataSource 和 SQLServerXADataSource。

如需詳細資訊，請參閱＜包裝函式與介面＞。
