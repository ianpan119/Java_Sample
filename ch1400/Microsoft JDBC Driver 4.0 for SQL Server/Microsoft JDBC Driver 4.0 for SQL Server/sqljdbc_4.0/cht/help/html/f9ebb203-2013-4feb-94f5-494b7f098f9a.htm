名稱

方法

名稱

描述

getParameterClassName

擷取 Java 類別的完整名稱，此類別的執行個體應該要傳遞給 SQLServerPreparedStatement 類別的 setObject 方法。

getParameterCount

擷取 SQLServerPreparedStatement 物件中參數的數目，而這個 SQLServerParameterMetaData 物件會包含前述物件的資訊。

getParameterMode

擷取所指定參數的模式。

getParameterType

擷取所指定參數的 SQL 類型。

getParameterTypeName

擷取所指定參數的資料庫特性類型名稱。

getPrecision

擷取所指定參數的小數位數。

getScale

擷取所指定參數小數點右邊的位數。

isNullable

擷取值，此值指出所指定參數中是否允許使用 Null 值。

isSigned

擷取值，此值指出所指定參數的值是否可以為帶正負號的數值。
