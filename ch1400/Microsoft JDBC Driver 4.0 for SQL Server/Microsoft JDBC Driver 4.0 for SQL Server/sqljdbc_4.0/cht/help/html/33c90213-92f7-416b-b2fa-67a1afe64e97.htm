start 方法 (SQLServerXAResource)

代表 Xid 物件中指定的交易分支開始工作。

語法

public void start(javax.transaction.xa.Xid xid,
                  int flags)

參數

xid

Xid 物件。

flags

int 值。

例外狀況

javax.transaction.xa.XAException

備註

這個 start 方法是由 javax.transaction.xa.XAResource 介面中的 start 方法所指定。
