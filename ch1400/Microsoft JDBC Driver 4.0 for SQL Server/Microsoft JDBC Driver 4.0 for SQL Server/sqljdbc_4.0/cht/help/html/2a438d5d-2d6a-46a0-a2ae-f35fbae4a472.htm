SQLServerResultSet 成員

下表列出由 SQLServerResultSet 類別公開的成員。

建構函式

無。

欄位

名稱

描述

CONCUR_SS_OPTIMISTIC_CC

用來指定不含資料列鎖定的 SQL Server 讀取/寫入開放式並行類型。

CONCUR_SS_OPTIMISTIC_CCVAL

用來指定不含資料列鎖定的 SQL Server 讀取/寫入開放式並行類型。

CONCUR_SS_SCROLL_LOCKS

用來指定包含資料列鎖定的 SQL Server 讀取/寫入開放式並行類型。

TYPE_SS_DIRECT_FORWARD_ONLY

用來指定 SQL Server 的順向和唯讀等資料指標類型。

TYPE_SS_SCROLL_DYNAMIC

用來指定 SQL Server 的動態資料指標類型。

TYPE_SS_SCROLL_KEYSET

用來指定 SQL Server 的索引鍵集資料指標類型。

TYPE_SS_SCROLL_STATIC

用來指定 SQL Server 的靜態資料指標類型。

TYPE_SS_SERVER_CURSOR_FORWARD_ONLY

用來指定 SQL Server 的順向和唯讀等資料指標類型。
