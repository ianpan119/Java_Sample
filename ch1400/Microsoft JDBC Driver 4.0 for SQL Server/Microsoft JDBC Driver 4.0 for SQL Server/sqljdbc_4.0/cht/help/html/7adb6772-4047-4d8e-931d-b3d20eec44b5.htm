rollback 方法 ()

復原目前交易中所做的所有變更，並釋放目前由這個 SQLServerConnection 物件所保留的所有資料庫鎖定。

語法

public void rollback()

例外狀況

SQLServerException

備註

這個 rollBack 方法是由 java.sql.Connection 介面中的 rollBack 方法所指定。

只有當已經停用自動認可模式時，才應該使用這個方法。
