getBytes 方法 (SQLServerResultSet)

使用 Java 程式語言，擷取這個 SQLServerResultSet 物件中目前資料列內指定之資料行的值來當做 byte 陣列。

多載清單

名稱

說明

getBytes (int)

使用 Java 程式語言，擷取這個 SQLServerResultSet 物件中目前資料列內指定之資料行索引的值來當做 byte 陣列。

getBytes (java.lang.String)

使用 Java 程式語言，擷取這個 SQLServerResultSet 物件中目前資料列內指定之資料行名稱的值來當做 byte 陣列。

備註

在舊版的 Microsoft JDBC Driver for SQL Server 中，您可以使用 SQLServerResultSet.getBytes 來轉換位元組陣列與 SQL Server 資料類型 date、time、datetime2 或 datetimeoffset 之間的值。現在，搭配這些資料類型使用這個方法將會引發例外狀況，指出不支援轉換。
