使用結果集

當您使用 SQL Server 資料庫中包含的資料時，一種操作資料的方法就是使用結果集。Microsoft JDBC Driver for SQL Server 支援透過 SQLServerResultSet 物件來使用結果集。藉由使用 SQLServerResultSet 物件，您可以擷取從 SQL 陳述式或預存程序傳回的資料、依需要更新資料，然後將該資料存回資料庫。 

此外，SQLServerResultSet 物件也提供數種方法，可用來瀏覽其資料列、取得或設定其包含的資料，以及對基礎資料庫中的變更建立各種敏感性層級。

如需管理結果集的詳細資訊 (包括其變更的敏感性)，請參閱＜使用 JDBC 驅動程式管理結果集＞。

本節的主題說明您可以使用結果集，操作 SQL Server 資料庫中所包含資料的不同方法。
