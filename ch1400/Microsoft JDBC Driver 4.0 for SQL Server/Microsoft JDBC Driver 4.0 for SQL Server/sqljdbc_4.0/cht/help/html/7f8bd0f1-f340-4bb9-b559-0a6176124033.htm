getCatalogs 方法 (SQLServerDatabaseMetaData)

擷取已連接伺服器中所提供的目錄名稱。

語法

public java.sql.ResultSet getCatalogs()

傳回值

SQLServerResultSet 物件。

例外狀況

SQLServerException

備註

這個 getCatalogs 方法是由 java.sql.DatabaseMetaData 介面中的 getCatalogs 方法所指定。

在 SQL Azure 上，您應該連接至 master 資料庫來呼叫 SQLServerDatabaseMetaData.getCatalogs。SQL Azure 不支援從使用者資料庫傳回整組目錄。SQLServerDatabaseMetaData.getCatalogs 會使用 sys.databases 檢視表取得目錄。請參考 sys.databases (SQL Azure 資料庫) 中的權限討論內容，以了解 SQL Azure 的 SQLServerDatabaseMetaData.getCatalogs 行為。

由 getCatalogs 方法傳回的結果集將包含下列資訊：

名稱

類型

描述

TABLE_CAT

String

目錄名稱，包括 Microsoft SQL Server 中的系統資料庫。

範例

在下列範例中，會說明如何使用 getCatalogs 方法來傳回 Microsoft SQL Server 中所包含任何資料庫的名稱，包括系統資料庫。

public static void executeGetCatalogs(Connection con) {
   try {
      DatabaseMetaData dbmd = con.getMetaData();
      ResultSet rs = dbmd.getCatalogs();
      ResultSetMetaData rsmd = rs.getMetaData();

      // Display the result set data.
      int cols = rsmd.getColumnCount();
      while(rs.next()) {
         for (int i = 1; i <= cols; i++) {
            System.out.println(rs.getString(i));
         }
      }
      rs.close();
   } 

   catch (Exception e) {
      e.printStackTrace();
   }
}
