SQLServerXAConnection 類別

代表可以參與分散式 (XA) 交易的 JDBC 連接。

封裝：com.microsoft.sqlserver.jdbc

擴充：SQLServerPooledConnection

實作：javax.sql.XAConnection

語法

public class SQLServerXAConnection

備註

SQLServerXAConnection 物件可以透過 SQLServerXAResource 物件編列在分散式交易中。通常屬於中間層伺服器之一部分的交易管理員會透過 SQLServerXAResource 物件管理 SQLServerXAConnection 物件。

應用程式設計人員通常不會直接使用這個介面。它主要是由在中間層伺服器工作的交易管理員所使用。
