recover 方法 (SQLServerXAResource)

從資源管理員取得備妥的交易分支清單。

語法

public javax.transaction.xa.Xid[] recover(int flags)

參數

flags

int 值，可以採用下列其中一個值：XAResource.TMSTARTRSCAN 或 XAResource.TMENDRSCAN 或 XAResource.TMNOFLAGS 或 XAResource.TMSTARTTRSCAN | XAResource.TMENDRSCAN。

傳回值

Xid 物件。

例外狀況

javax.transaction.xa.XAException

備註

這個 recover 方法是由 javax.transaction.xa.XAResource 介面中的 recover 方法所指定。

如果 flag 參數不是 XAResource.TMSTARTRSCAN 或 XAResource.TMSTARTRSCAN | XAResource.TMENDRSCAN，則表示復原掃描必定正在進行中。
