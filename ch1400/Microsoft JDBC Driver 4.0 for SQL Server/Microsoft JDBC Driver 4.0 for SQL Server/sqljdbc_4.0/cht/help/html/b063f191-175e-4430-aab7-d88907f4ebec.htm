繼承的欄位

方法

名稱

描述

free

這個方法會釋放 NCLOB 物件並且釋出它所保留的資源。

getAsciiStream

擷取由 java.sql.NClob 物件指定的 NCLOB 值當做 ASCII 資料流。

getCharacterStream

擷取由 java.sql.NClob 物件指定的 NCLOB 值。

getSubString

擷取由 java.sql.NClob 物件所指定 NCLOB 值中的指定子字串副本。

length

擷取由 java.sql.NClob 物件所指定 NCLOB 值中的字元數目。

position

依據指定的開始位置，擷取所指定 java.sql.NClob 物件或 java.sql.NClob 中之子字串的字元位置。

setAsciiStream

擷取資料流，此資料流將用於從指定位置開始，將 ASCII 字元寫入到這個 java.sql.NClob 物件所代表的 NCLOB 值。

setCharacterStream

擷取資料流，此資料流將用於從指定位置開始，將 Unicode 字元資料流寫入到這個 java.sql.NClob 物件所代表的 NCLOB 值。

setString

從指定位置開始，寫入指定的 String 到 NCLOB。

truncate

截斷 NCLOB 值成為指定的長度。
