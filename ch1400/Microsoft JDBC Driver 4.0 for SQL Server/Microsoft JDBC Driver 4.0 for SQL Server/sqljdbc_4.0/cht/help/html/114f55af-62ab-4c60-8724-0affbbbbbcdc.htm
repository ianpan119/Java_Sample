valueOf 方法 (java.sql.Timestamp, int)

建立 DateTimeOffset 物件，假設是 java.sql.Timestamp 值以及表示指出位移 (以分鐘為單位) 的值，這個物件表示從 GMT 特定位移的時間點。

語法

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, int minutesOffset)

參數

timestamp

java.sql.Timestamp 值。

minutesOffset

以分鐘為單位的位移。

傳回值

傳回 DateTimeOffset 物件，這個物件表示 java.sql.Timestamp 物件在 GMT 給定位移 (以分鐘為單位) 提供的時間點。
