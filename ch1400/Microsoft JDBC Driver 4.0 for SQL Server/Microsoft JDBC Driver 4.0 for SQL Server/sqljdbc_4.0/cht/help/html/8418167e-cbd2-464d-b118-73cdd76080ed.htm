executeBatch 方法 (SQLServerPreparedStatement)

將命令批次提交到要執行的資料庫。如果所有命令都成功執行，則傳回更新計數陣列。

語法

public int[] executeBatch()

傳回值

包含更新計數的 int 陣列。

例外狀況

SQLServerException

java.sql.BatchUpdateException

備註

這個 executeBatch 方法是由 java.sql.Statement 介面中的 executeBatch 方法所指定。

SQL Server JDBC 驅動程式 3.0 與 JDBC 4.0 建議相容，該建議指出呼叫 CallableStatement.executeBatch 方法 (繼承自 PreparedStatement) 將會擲回 BatchUpdateException，前提是預存程序接受 OUT 或 INOUT 參數或傳回更新計數以外的項目。

這個方法會覆寫 SQLServerStatement.executeBatch。
